# Smart Book Navigator

## Description

Smart Book Navigator is a simple library navigation system that helps users search and locate books easily.

## Features

* Search books by title or author
* Simple and user-friendly interface
* Quick navigation

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Clone the repository
2. Open the project folder
3. Run `index.html` in browser
